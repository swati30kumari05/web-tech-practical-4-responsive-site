# Practical 4 – Responsive One-Page Website

This repository contains my Web Technologies Practical‑4: a fully responsive one‑page website built with HTML, CSS, and a small amount of JavaScript. The page follows the given reference layout with a large hero banner, team members, project overview, technologies used, and a simple contact section.[web:69][web:81]

## Features

- Large hero image with overlay title “Practical – 4”
- Team section with four circular employee avatars
- About Us section with full text on the left and three project preview images on the right
- “Technologies Used” cards for HTML, CSS, JavaScript, and Bootstrap with card images and Read More buttons
- Contact strip with footer “Back to top” link
- Responsive navigation bar:
  - Desktop: full horizontal menu
  - Tablet & mobile: hamburger icon with full‑screen slide‑in menu[web:69][web:79]

## Technologies

- HTML5 for structure
- Modern CSS (Flexbox, Grid, media queries)
- Vanilla JavaScript for the mobile navigation toggle
